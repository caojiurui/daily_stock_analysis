# News Catalyst Aggregation

聚合新闻、题材驱动、热点路线和机会引擎中的催化证据，判断“最近发生了什么”和“是否足以形成可跟踪机会”。

## Usage

```text
/news-catalyst-aggregation <topic_or_symbol>
```

## Instructions

分析时使用简洁中文，优先遵循仓库根目录 `AGENTS.md`。

### Step 1: 聚合多源催化

优先使用：

- `key_news`
- AlphaSift 热点 route
- THS 题材驱动事件
- 现有搜索新闻结果

### Step 2: 做 3 个判断

- 是否是新催化，而不是旧消息重复
- 是否能映射到明确板块 / 概念 / 标的
- 是否已经形成多源共振，而不是单条噪音

### Step 3: 输出催化等级

可用：

- `强催化`
- `一般催化`
- `仅作背景`
- `噪音/不纳入`

### Step 4: 降噪原则

- 标题党、转载聚合、时间过旧、无法映射到主题的消息，优先降级
- 若来源之间互相矛盾，明确写出冲突，不要强行合并成单结论

## Output Format

```markdown
## 催化聚合结论

- 催化等级：
- 主要事件：
- 关联主题：
- 是否形成机会：
- 风险与噪音：
```
